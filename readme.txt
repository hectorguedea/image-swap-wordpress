== Description ==

Images Swap is a dynamic WordPress plugin that introduces a new level of interactive design to your website, allowing users to swap images or backgrounds effortlessly. Compatible with any theme or page builder, including Elementor, Beaver Builder, Divi, and more, Images Swap is the perfect tool for enhancing the visual appeal and engagement of your site.

**Features Include:**
- **Universal Compatibility**: Works seamlessly with any WordPress theme or page builder, providing flexibility and ease of integration.
- **Dynamic Content Interaction**: Enable image or background swaps from links, buttons, or accordions, adding an interactive element to your site.
- **Customizable Transitions**: Elevate the user experience with a selection of transition effects, adjustable to fit your site's aesthetic.
- **Advanced Customization Options**: Utilize custom CSS classes to fine-tune the appearance and behavior of your swappable elements.

Ideal for website owners, developers, and designers looking to add an interactive and engaging layer to their web pages.

== Installation ==

1. Download the `images-swap` zip file from the Plugin URI.
2. Go to your WordPress dashboard, navigate to 'Plugins' > 'Add New', and click on 'Upload Plugin' at the top of the page.
3. Choose the downloaded zip file and click 'Install Now'.
4. Once installed, click 'Activate Plugin' to start using Images Swap on your site.

== Usage ==

1. After activation, head over to `Tools` > `Images Swap` to access the plugin settings.
2. Define CSS classes for your image swap triggers such as links, buttons, or accordion items to match your site's design.
3. Choose from a variety of transition effects and durations to create smooth and visually appealing image transitions.
4. Apply your configured CSS classes within your site's elements to enable the image swapping feature.
5. Ensure that the IDs of your triggers correspond to the names of the images in your Media Library for flawless functionality.

== Frequently Asked Questions ==

= Can I specify different transition effects for different elements? =

Absolutely! You can assign unique CSS classes to different elements and select distinct transition effects for each, providing you with complete control over the animations.

= Do I need to write any code to use this plugin? =

No coding is required! Images Swap is designed for ease of use, with a straightforward settings interface that lets you configure everything without touching a single line of code.
